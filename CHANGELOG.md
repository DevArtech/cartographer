## [0.1.0] - Initial Release

### ✨ Features

- Network device discovery and mapping
- Interactive drag-and-drop network visualization
- Real-time health monitoring with status indicators
- AI assistant for natural language network queries
- Multi-user support with authentication
- Email and Discord notifications
- Anomaly detection for unusual network behavior
- Embeddable network maps
- Auto-save functionality
